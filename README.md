# 🚀 Immersive 3D Creative Portfolio

A high-performance, interactive 3D desk environment built to showcase my transition from 3D modeling to full-stack web development.

## 🔗 Live Experience
**[Visit Live Site: portfolio-project-tmok.vercel.app](https://portfolio-project-tmok.vercel.app)**

---

## ✨ Project Highlights
* **Interactive 3D Scene:** Custom-modeled desk environment built in **Blender**.
* **Scroll-Linked Animation:** Seamless camera transitions using `ScrollControls` for a cinematic feel.
* **Dynamic Scaling:** Custom-built responsive engine to ensure 3D elements and UI overlays align perfectly across all devices.
* **Dual Theme Support:** Real-time toggle between **Dark** and **Light** modes.

## 🛠️ Technical Stack
* **Core:** React.js + Vite
* **3D Engine:** Three.js (React Three Fiber & Drei)
* **Modeling:** Blender (Optimized `.glb` assets)
* **Deployment:** Vercel

## 📱 Viewing Note
This portfolio is an immersive experience. For **mobile users**, please rotate your device to **Landscape Mode** and click the **Fullscreen** button for the best visual fidelity.
